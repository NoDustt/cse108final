CSE 108 Final Game

This game is a varation of a Tic Tac Toe, consisting of a 3x3 grid.
Each spot contains another 3x3 grid where an instance of Tic Tac Toe is played.
The starting player can select any square to play on
The player may choose any empty spot in one of the higlighted 3x3 grid.
If a player wins on one of the 3x3 grids, the grid now changes into the player's color.
If a player gets 3 big markers in a row, the player wins.
